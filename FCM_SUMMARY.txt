### ✅ CORREÇÕES APLICADAS

#### 1. Deep Linking (MainActivity + AppNavigation)
```kotlin
// ✅ NOVO em MainActivity
val receiverId = intent.getStringExtra("receiverId")
AppNavigation(deepLinkReceiverId = receiverId)

// ✅ NOVO em AppNavigation
LaunchedEffect(deepLinkReceiverId) {
    if (deepLinkReceiverId != null && authUiState.token != null) {
        val userId = deepLinkReceiverId.toIntOrNull()
        if (userId != null) {
            navController.navigate(Screen.Chat.createRoute(userId))
        }
    }
}
```

#### 2. Validação de Permissões (Android 13+)
```kotlin
// ✅ NOVO em FirebaseMessagingService
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.TIRAMISU) {
    if (ContextCompat.checkSelfPermission(
        this,
        android.Manifest.permission.POST_NOTIFICATIONS
    ) != PackageManager.PERMISSION_GRANTED) {
        Log.w("FCM", "POST_NOTIFICATIONS not granted")
        return
    }
}
```

#### 3. Normalizar Payload
```typescript
// ✅ ANTES:
data: { receiverId, type: 'new_message' }

// ✅ DEPOIS:
data: { type: 'new_message', receiverId }

// ✅ Android lê corretamente:
remoteMessage.data["receiverId"]
```
